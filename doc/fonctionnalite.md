# Stone Starter PHP

Starter ultra léger pour PHP avec moteur Blade-like inspiré de StoneFw.

## Fonctionnalités

- `View::render('template', $data)`
- Supporte `@extends`, `@section`, `@yield`, `@include`
- Supporte `@foreach`, `@if`, `@for`, `@while`, etc.
- Toutes les variables passées sont accessibles dans les sections et includes
- Cache automatique pour les vues compilées
